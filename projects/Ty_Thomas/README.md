Project Overview
This project is a 30-second ambience-focused resound of a scene from *Resident Evil Village*.  
The goal was to recreate environmental tension and atmosphere through layered ambience, foley, and controlled mixing using REAPER.

 Tools Used
- REAPER (sound design and mixing)
